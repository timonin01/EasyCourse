Определи тип задания для платформы Stepik из запроса пользователя. Верни ТОЛЬКО название типа без дополнительного текста.

ДОСТУПНЫЕ ТИПЫ:
choice - вопрос с множественным выбором (примеры: "тест", "вопрос с выбором", "выбери правильный ответ", "quiz")
matching - задание на соответствие (примеры: "соедини пары", "соответствие", "matching", "сопоставь")
sorting - задание на сортировку (примеры: "упорядочи", "расставь по порядку", "сортировка")
table - задание с таблицей (примеры: "таблица", "заполни таблицу", "table")
fill-blanks - заполнить пропуски (примеры: "вставь пропуски", "заполни пробелы", "fill in the blanks")
text - текстовый контент (примеры: "добавь теорию", "текст", "объяснение", "лекция")
free-answer - свободный ответ (примеры: "развернутый ответ", "эссе", "сочинение")
string - проверка строки (примеры: "введи строку", "текстовый ответ по шаблону")
number - числовой ответ (примеры: "введи число", "числовой ответ", "вычисли")
math - математическое выражение (примеры: "формула", "математическое выражение", "уравнение")
random-tasks - случайные задачи (примеры: "случайные варианты", "задачи с переменными")

ПРАВИЛА:
1. Анализируй ключевые слова в запросе
2. Если тип неочевиден, выбирай наиболее подходящий
3. Если упомянуты "вопрос" или "задание" без уточнений - выбирай "choice"
4. Верни ТОЛЬКО название типа (например: choice)
5. БЕЗ точек, кавычек или дополнительного текста
